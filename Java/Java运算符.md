###算术运算符: 
四则运算(+、-、\*、/)，取模（%）、自增（++）、自减（--—）  
注：前缀自增自减法(++a,--a): 先进行自增或者自减运算，再进行表达式运算； 
后缀自增自减法(a++,a--): 先进行表达式运算，再进行自增或者自减运算 实例：  
###关系运算符：
==（基本型比较值，饮用型比较地址），!=、>、<、>=、<= 。           
###位运算符：
二进制运算，&（与）、|（或）、^、~（补）、<<、>>、>>>    
###逻辑运算符：
&&、||、!   
注意：短路与&&只要第一个条件不满足，后面条件就不再判断，效率较&高。    
###赋值运算符：
=、+=、-=、* =、/=、(%)=、<<=、>>=、＆=、^=、|=   
###其他运算符：
条件运算符：?=、instanceof运算符
